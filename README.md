# demo-valencia-pericial-permed

Demo conceptual no oficial para PERMED.

Sector: peritos judiciales y mediadores
Ciudad: Valencia / Alicante / Castellon
Fuente: https://www.permed.es/index.php/es/garantia/garantia
Contacto publico: permed@permed.es
Precio propuesto: 780 EUR

Punto debil trabajado:
Servicios tecnicos de alto valor quedan mezclados con plantilla antigua, comentarios viejos y textos largos que reducen percepcion de precision.

Stack: HTML estatico, CSS embebido y vercel.json para despliegue en Vercel.
